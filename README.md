# System-Manager
Node based tool to view a servers system-stats through a webpage

**Version 1.0.0**

## Installation / Execution

1. Install Node js
2. Install the dependencies (see below)
3. Type node index.js into the command prompt

## Dependencies

- node-os-utils

Commands to install:

> npm install node-os-utils
